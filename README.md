## Run Locally

Clone the project

```bash
  git clone https://github.com/DominikReszczynski/Pokedex_TS.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

**Client:** TypeScript, React, Redux, SCSS
